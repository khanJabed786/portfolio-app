# Portfolio App (Jabed Khan)

Modern portfolio built with **React + Vite + Tailwind + Firebase**.

## Features
- Full-screen sections + sticky navbar + smooth scroll
- Animated gradient background + particles + custom cursor
- Typing effect + scroll animations (AOS)
- Projects: search/filter + video modal + gallery lightbox
- Contact form saved to Firestore
- Admin panel (email allowlist) for editing content and viewing messages/analytics
- PWA offline cache (basic)
- SEO: meta tags + sitemap + robots
